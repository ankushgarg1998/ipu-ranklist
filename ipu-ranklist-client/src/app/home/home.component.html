IPU Ranklist

A minimalist Result and Ranklist application for IP University.

The application covers the following results of all batches since 2015 :

- B. Tech. results of 18 colleges including all thirteen branches of engineering.

- BCA results of 25 colleges.

- BBA result of 42 colleges including three branches (General, B&I and CAM).

This project is public and you can track its progress on the GitHub Repo.

If you find any bugs, or would like suggest some features, or have any questions regarding IPU Ranklist, feel free to drop a mail at admin@ipuranklist.com or raise an issue.
